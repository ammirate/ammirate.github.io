MAEESTRO is a proof of concept developed during my Bachelor's internsheep at Aicas, that aims to run Android code on any Java Virtual Machine.
The idea comes from the possibility to use the JamaicaCar Virtual Machine, a custom JVM for embedded devices, for running your favorite apps in your infotainment system.
MAEESTRO can execute several types of app, including widgets and simple games, in different resolutions and screen orientations.
