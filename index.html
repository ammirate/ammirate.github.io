On the left the app emulated by MAEESTRO, on the right the original one.


MAEESTRO is a proof of concept realized during my Bachelor's internsheep at Aicas, which aims to run Android code on any Java Virtual Machine. The idea came from the chance to use a JVM for embedded device, the Jamaica Car Virtual Machine, and it was very exiting to have car devices able to run the same app that you have in your smartphone.
It is able to run many different types of apps, including widgets and simple games too, in different resolutions and screen orientation.
